# Build Phases

## Phase 1: Foundation and monetization core

- Tenant onboarding
- Auth and roles
- SaaS subscription plans
- Operator dashboard shell
- Customer CRUD
- Package CRUD
- Payment collection
- Receipts

## Phase 2: Operations core

- Due engine
- Recharge workflows
- Staff permissions
- Area and collector assignment
- Complaints
- Expenses
- Reminder scheduler

## Phase 3: Scale and integrations

- External billing adapters
- CSV import pipeline
- Reconciliation console
- Operator analytics
- Customer portal
- Webhook center
